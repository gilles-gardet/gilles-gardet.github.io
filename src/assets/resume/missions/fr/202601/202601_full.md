- Mise en place d’un registre innovant pour la gestion des serveurs MCP au sein d’Air France KLM.  
- Conception et implémentation d’une solution robuste adaptée aux contraintes métier et techniques :  
  - Architecture modulaire en microservices avec SpringBoot et Spring Data.  
  - Interface utilisateur développée en Angular 20 pour une expérience fluide.  
  - Orchestration des workflows via Camunda pour automatiser les processus.  
- Rédaction d’une spécification technique complète, détaillant toutes les exigences fonctionnelles et techniques.  
- Collaboration étroite avec l’équipe multidisciplinaire : échanges constants avec le product owner et le business analyst pour aligner la solution sur les besoins métier.  
- Utilisation de Copilot pour accélérer le développement et améliorer la qualité du code.  
- Livraisons incrémentales suivant les sprints Scrum, avec revues régulières et ajustements.  
**Technologies utilisées :**  
```
Java 21, Angular 20, Maven, Spring Data, SpringBoot, Camunda, Copilot
```
