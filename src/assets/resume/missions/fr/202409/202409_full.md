- Conception et développement de microservices backend pour la gestion du catalogue produits, des commandes et des paiements  
  - Implémentation d’API REST en Kotlin avec Spring Boot 3 suivant les bonnes pratiques DDD et TDD  
  - Création de workflows asynchrones avec RabbitMQ pour le traitement des commandes et notifications  
  - Mise en place d’un système de gestion des stocks en temps réel avec Redis pour garantir la cohérence des données  
- Déploiement et orchestration cloud-native  
  - Conception de l’infrastructure Kubernetes pour un déploiement scalable et résilient des microservices  
  - Containerisation des applications avec Docker, gestion des configurations et secrets sécurisés  
- Développement frontend React pour une interface utilisateur moderne et réactive  
  - Intégration fluide avec les API backend et gestion optimisée des états locaux et globaux  
  - Mise en place d’un système d’authentification sécurisé et gestion des paiements via API tierces  
- Collaboration en équipe agile et amélioration continue  
  - Participation active aux sprints de 2 semaines, revues de code et sessions de planification  
  - Mise en place de pipelines CI/CD automatisés pour accélérer le déploiement et la qualité  

**Technologies utilisées :**
```
Kotlin, Spring Boot 3, Kubernetes, PostgreSQL, React, Docker, RabbitMQ, Redis
```
