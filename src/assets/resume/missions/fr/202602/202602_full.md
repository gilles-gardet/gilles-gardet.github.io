Conception et développement du registre MCP pour centraliser la gestion des serveurs.  
  - Mise en place d’une architecture basée sur **SpringBoot** et **Spring Data** pour la gestion des données relationnelles.  
  - Intégration de processus automatisés avec **Camunda** pour la gestion des workflows métier.  
  - Implémentation d’une interface utilisateur en **Angular** pour faciliter l’accès aux données serveur.  

Collaboration étroite avec le Product Owner et l’analyste business pour définir les besoins fonctionnels.  
  - Définition des cas d’usage pour la création, la mise à jour et la suppression des entrées serveur.  
  - Validation continue des spécifications techniques avec le client.  
  - Adaptation agile aux retours utilisateurs pour ajuster les fonctionnalités.  

Automatisation du déploiement et gestion de l’infrastructure sur **AKS**.  
  - Configuration de pipelines CI/CD via **Github Actions** pour assurer les builds et tests automatiques.  
  - Déploiement orchestré avec **Ansible** sur un cluster Kubernetes.  
  - Surveillance et gestion des performances post-déploiement.  

```
Java 21, Camunda, Angular, SpringBoot, Spring Data, Github Actions, Ansible, AKS
```
