  • Technical achievements:
    • Implementation of Spring Batch jobs to orchestrate ETL processes.
    • Integration of a "real-time" module continuously fetching external data.
    • Setup of HiveMQ streams for real-time data push.
    • Development of Angular UI for dashboard visualization and customization.
  • Architecture:
    • SpringBoot framework to structure the API.
    • Primary PostgreSQL database supported by MongoDB for specific data storage needs.
    • Deployment on Azure Kubernetes Service to ensure scalability and resilience.
  • Collaboration:
    • Agile Kanban team, with ongoing communication between PO and developers.
  • Data management:
    • Control-M scripts ensuring secure scheduling and execution of batches.
